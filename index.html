End-to-End Tests

Last E2E test was completed at:

The E2E tests are configured as a dedicated github actions workflow in the local-deploy, which is triggerd by every change in the repository.

The current test-scenario completes the following steps:

  • Deploying the whole system based on the docker-compose configuration.
  • Fetching adaptors from the manager to verify registration of adaptors.
  • Registering simulation instances for webots and demo adaptor.
  • Fetching simulation instances to verify registration of instances.
  • Initializing a session wtih demo and webots adaptors.
  • Sending a set-joint-position command for the session.
  • Verifying the responses from the simulations for the executed command.
